net-macaddress
==============

Description
------------

Utils for MACaddresses

Features
--------

* Usuful methods
* Immutable

Usage
-----

### Setup

```ruby
require 'net/macaddress'
include Net
```

Requirements
------------

* [Ruby 2.5 or later](http://travis-ci.org/#!/kachick/net-macaddress)

Installation
-------------

```shell
$ gem install net-macaddress
```
